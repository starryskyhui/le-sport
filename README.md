# le-sport

## 项目介绍
le-sport （爱运动）是一个前后端分离的**体育消费券免费发放及在线购票**项目，用户可以在页面内免费抢消费券，并在线购买体育馆的入场票，在购票过程中可以使用优惠券进行优惠，并使用微信支付进行支付。
